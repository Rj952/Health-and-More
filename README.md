# ROOTS — Caribbean Healthy Ageing Platform

A mobile-first PWA for Caribbean seniors, connecting health tracking, family caregiving, government benefits navigation, and cultural wellness.

## Features
- **Health Companion** — BP tracking, medication management, symptom logging
- **Family Bridge** — Diaspora family connection with voice notes
- **Benefits Navigator** — NHF, JADEP, NIS, PATH programme guides for 12 CARICOM nations
- **Cultural Wellness** — Devotionals, proverbs, bush remedies, story recording
- **WhatsApp Integration** — Share health updates directly via WhatsApp
- **WCAG 2.1 AA Accessible** — Screen reader support, keyboard navigation, reduced motion

## Tech
Single-file HTML/CSS/JS • No build step • Mobile-first responsive • Works offline-ready

## Deploy
